==超速決鬥中的帝==
* 首次登場於2023年5月推出的卡包「閃光之忘卻」，但以新怪獸加入超速決鬥
* 後期在不同的補充包慢慢加入原本的六式帝，並在2025年9月推出專為帝王而設的預組
* 和原版不同，整個系列的共通體質為「守備力1000」
** 超速決鬥原創怪獸攻擊力只有2000攻擊力，進化形態才達到原本的2400攻擊力
** 後期加入的原版回到原本的2400攻擊力，但原效果放在超速決鬥也太逆天而全員被列入傳說卡

== 牌組怪獸介紹 ==

=== 賢帝系列 ===
* 收錄於補充包「閃光之忘卻」，超速決鬥加入的新成員之一，也是電子族第一張帝系列
* 雖然有著破壞卡片的能力，但由於不用代價，效果綁了一堆限制。
<br>
==== 賢帝家臣 楚澤（賢帝家臣ツーゼ） ====
* 【賢帝】的支援卡片。
* 其效果可以用作回收帝王系列卡片。
===== 卡片資訊 =====
* 效果怪獸<br>等級3／暗屬性／電子族／攻擊力800／守備力1000
* 效果：
**【條件】此卡召喚的回合，將我方2張守備力1000的怪獸從墓地回到牌組才能發動
**【效果】對方將自身墓地魔法卡1張在自己場上覆蓋。之後，我方可以將我方墓地怪獸(等級5以上/守備力1000)1體加入手牌
{{-}}

==== 賢帝 威廉（賢帝ヴィルヘル） ====
* 效果可確定對方卡片，但只能破壞魔法卡，不能破壞陷阱卡。
===== 卡片資訊 =====
* 效果怪獸<br>等級5／暗屬性／電子族／攻擊力2000／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】確認對方場上裡側表示魔法˙陷阱卡1張。該卡是魔法卡的場合，可以再將該卡破壞
{{-}}

==== 才賢帝 威廉（才賢帝ヴィルヘル） ====
* 【賢帝】的進化版。
* 效果可破壞任何卡片，但要由對方選擇。
===== 卡片資訊 =====
* 效果怪獸<br>等級8／暗屬性／電子族／攻擊力2400／守備力1000
* 效果：
** 可以只釋放1隻等級5以上守備力1000的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合才能發動。
**【效果】我方讓對方決定其自身場上1張卡破壞
{{-}}

=== 星帝系列 ===
* 收錄於補充包「閃光之忘卻」，超速決鬥加入的新成員之一，也是銀河族第一張帝系列
* 和【賢帝】一樣有著破壞卡片的能力，但由於不用代價，效果綁了一堆限制。
<br>
==== 星帝家臣 赫貝（星帝家臣ヘーベ） ====
* 為【星帝】的支援卡片。
* 其效果可以用作回收帝王系列卡片，同時召喚足夠素材召喚帝皇。
===== 卡片資訊 =====
* 效果怪獸<br>等級3／光屬性／銀河族／攻擊力800／守備力1000
* 效果：
**【條件】此卡召喚的回合，將我方2張守備力1000的怪獸從墓地回到牌組才能發動
**【效果】對方將自身墓地1隻等級6以下的效果怪獸在自己場上表側守備表示特殊召喚。之後，我方可以將我方墓地1隻等級5以上和守備力1000怪獸加入手牌
{{-}}

==== 星帝 艾斯托姆（星帝エストローム） ====
* 效果可破壞對方怪獸，但只限等級6或下的效果怪獸，代表不能破壞等級6以上，或是任何通常怪獸。
===== 卡片資訊 =====
* 效果怪獸<br>等級5／光屬性／銀河族／攻擊力2000／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】破壞對手場上1隻第級6以下的效果怪獸。
{{-}}

==== 煌星帝 艾斯托姆（煌星帝エストローム） ====
* 【星帝】的進化版。
* 效果可破壞任何卡片，但要由對方選擇。
===== 卡片資訊 =====
* 效果怪獸<br>等級8／光屬性／銀河族／攻擊力2400／守備力1000
* 效果：
** 可以只釋放1隻等級5以上守備力1000的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合才能發動。
**【效果】我方讓對方決定其自身場上1張卡破壞
{{-}}

=== 龍帝系列 ===
* 收錄於預組牌組「帝王的凱旋」，超速決鬥加入的新成員之一，也是幻龍族第一張帝系列
<br>

==== 龍帝 林德魯姆（竜帝リンドルム） ====
* 效果除了可以破壞對方堆墓外，也可以減少對方場上怪獸的攻擊力，而且效果的唯一限制是額外牌組沒有任何卡片，是新【帝王】中比較好用的一張。
===== 卡片資訊 =====
* 效果怪獸<br>等級5／水屬性／幻龍族／攻擊力2000／守備力1000
* 效果：
** 可以只釋放1隻等級5以上守備力1000的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合，而且我方額外牌組沒有卡時才能發動。
**【效果】對方墓地最多2張卡回到牌組。之後，可以讓對方場上表側表示效果怪獸1體這回合攻擊力˙守備力下降800
{{-}}

==== 幻龍帝 林德魯姆（幻竜帝リンドルム） ====
* 【龍帝】的進化版。
* 效果可令任卡片回到牌組，但要由對方選擇，而且多了額外牌組清零的條件
===== 卡片資訊 =====
* 效果怪獸<br>等級8／水屬性／幻龍族／攻擊力2400／守備力1000
* 效果：
** 可以只釋放1隻等級5以上守備力1000的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合，而且我方額外牌組沒有卡時才能發動。
**【效果】對方選自身場上2張卡，我方從中將1張回到持有者牌組。
{{-}}

=== 原六帝系列 ===
* 現時超速決鬥已引入《雷帝》《炎帝》《風帝》《地帝》《冰帝》《邪帝》
* 由於超速決鬥沒有召喚時誘發效果，因此全都改成效果只能在召喚回合才能發動。
* 由於每張的攻擊力高達2400，而且所有效果都沒有發動代價，甚至能夠達到一換一甚至一換二的效果，因此每張都被列入傳說卡
<br>
==== 雷帝 扎博爾格（{{ruby|雷帝|らいてい}}ザボルグ） ====
* 收錄於補充包KP13，為第一隻收錄的六帝，也是「六帝」怪獸中唯一一隻等級5怪獸。
* 由新效果不是召喚時必發，改善原版[[自爆]]的弱點。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級5／光屬性／雷族／攻擊力2400／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】破壞對手場上1隻怪獸。
{{-}}

==== 冰帝 梅比烏斯（{{ruby|氷帝|ひょうてい}}メビウス） ====
* 收錄於補充包「真・傳說覺醒包」。
* 效果和原版一樣，可以破壞場上0～2張魔陷。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／水屬性／水族／攻擊力2400／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】破壞對手場上最多2張魔法卡或陷阱卡。
{{-}}

==== 炎帝 泰斯塔羅斯（{{ruby|炎帝|えんてい}}テスタロス） ====
* 初登場為V Jump特典卡，後期正式收錄於補充包「5周年包」。
* 和原版一樣，隨機丟棄對手的一張手牌並造成傷害，但因遊戲機制很多時候對手跟本沒有手牌可以弄。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／炎屬性／炎族／攻擊力2400／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】對手隨機丟棄1張手牌。若丟棄了怪獸，則給予對手那隻怪獸的等級×100點的傷害。
{{-}}

==== 地帝 格蘭瑪格（{{ruby|地帝|ちてい}}グランマーグ） ====
* 收錄於補充包「覺醒霸王龍」。
* 和原版一樣，可以破壞一張蓋卡，不論是魔陷卡還是怪獸卡。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／地屬性／岩石族／攻擊力2400／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】破壞1張蓋牌。
{{-}}

==== 風帝 瑞薩（{{ruby|風帝|ふうてい}}ライザー） ====
* 初收錄於構築卡組「神鷹女郎姊妹」。
* 可以將場上一張卡片彈回牌組最上方。在原版為極強力的效果，但在超速決鬥因每回合可以抽多於一張所以影響較少。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／風屬性／鳥獸族／攻擊力2400／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】選擇場上1張卡牌回牌組上方。
{{-}}

==== 邪帝 蓋烏斯（{{ruby|邪帝|じゃてい}}ガイウス） ====
* 收錄於預組牌組「帝王的凱旋」
* 由於超速決鬥沒有除外區，因此效果改為回到卡組，但保留指定闇屬性怪獸就能向對手造成傷害的效果。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／闇屬性／惡魔族／攻擊力2400／守備力1000
* 效果：
**【條件】在召喚了這張卡的回合才能發動。
**【效果】選擇場上1張卡牌回牌組下方。若選擇闇屬性怪獸，則給予對手1000點傷害。
{{-}}

== 相關陷阱 ==

=== 上級衝擊（アドバンス・インパクト）===
* 收錄於補充包KP13，弱化版的《烈旋》
* 除了要求場上沒有怪獸，而且要以手牌作為代價才可以令對方怪獸成為自己的祭品。
===== 卡片資訊 =====
* 通常魔法
**【條件】我方場上沒有怪獸的場合可以發動
**【效果】將等級合計為對方場上表側表示怪獸1體的我方手牌怪獸送墓。這回合，我方上級召喚怪獸的場合，作為我方場上怪獸1體的代替可以解放對方表側表示的怪獸
{{-}}

=== 帝王領域（エンペラー・レルム）===
* 收錄於補充包KP13，弱化版的《真帝王領域》
* 效果被斬到只留下增加攻擊力效果，而且並不是永續效果，也是在RD中少數有著發動條件的場地魔法
* 卡圖表示《賢帝》和《星帝》為戰友關係<s>，難怪兩人效果如此相似</s>
===== 卡片資訊 =====
* 場地魔法
**【條件】此卡將牌組上方2張卡送墓可以發動
**【效果】此卡在場地區表側表示存在為限，雙方場上原守備力1000的表側表示怪獸(等級5以上)攻擊力上升400
{{-}}

=== 帝王的烈旋（帝王の{{ruby|烈旋|れっせん}}）===
* 收錄於預組牌組「帝王的凱旋」。
* 引入後變回通常，但效果複製原版的效果。
* 由於效果無任何代價，因此被列入傳說卡之列。
===== 卡片資訊 =====
* 通常魔法（LEGEND）
**【條件】我方沒有將額外牌組怪獸特殊召喚過的回合可以發動
**【效果】對方場上怪獸1體，這回合在我方將怪獸上級召喚的場合，作為我方場上怪獸1體的代替可以將該對方怪獸解放。此效果發動後，這回合我方不能從額外牌組特殊召喚
{{-}}

=== 越境的帝王（越境の帝王）===
* 收錄於預組牌組「帝王的凱旋」，回收墓地之用
* 跟據卡圖所示，RD的新帝王和OCG的舊帝玊存在於不同的時空，但一次意外，暗帝入侵了RD的時空...
===== 卡片資訊 =====
* 通常魔法
**【條件】無
**【效果】我方攻擊力2000或2400，守備力1000的怪獸1體從墓地加入手牌
{{-}}

=== 帝王的機才（帝王の機才）===
* 收錄於補充包KP13
* 圖中為《賢帝》
===== 卡片資訊 =====
* 通常陷阱
**【條件】我方場上有怪獸(等級5以上/守備力1000)的場合，對方召喚怪獸時可以發動
**【效果】該召喚的怪獸變成裡側守備表示。之後，可以將我方墓地怪獸(等級5以上/守備力1000)1體加入手牌
{{-}}

=== 帝王的煌誕（帝王の煌誕）===
* 收錄於補充包KP13
* 圖中為《星帝》
===== 卡片資訊 =====
* 通常陷阱
**【條件】對方表側表示召喚˙特殊召喚怪獸(等級8以下)時，將我方場上怪獸1體送墓可以發動
**【效果】手牌怪獸(等級5以上/守備力1000)1體在我方場上表側攻擊表示特殊召喚。之後，可以將對手該次召喚˙特殊召喚的怪獸中1體破壞
{{-}}
