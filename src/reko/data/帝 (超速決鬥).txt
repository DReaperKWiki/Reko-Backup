==超速決鬥中的帝==
* 首次登場於2023年5月推出的卡包「閃光的忘卻龍」，但一開始只有新的原創怪獸
* 後期在不同的補充包慢慢加入原本的六式帝，並在2025年9月推出專為帝王而設的預組
* 和原版不同，整個系列的共通體質為「守備力1000」
** 超速決鬥原創怪獸攻擊力只有2000攻擊力，進化形態才達到原本的2400攻擊力
** 後期加入的原版回到原本的2400攻擊力，但原效果放在超速決鬥也太強勢而全員被列入傳說卡

== 牌組怪獸介紹 ==

=== 賢帝系列 ===
* 收錄於補充包「閃光的忘卻龍」，超速決鬥加入的新成員之一，是網域族的成員。
* 雖然有著破壞卡片的能力，但由於不用代價，效果綁了一堆限制。
<br>
==== 賢帝家臣 楚澤（{{ruby|賢帝家臣|けんていかしん}}ツーゼ） ====
* 【賢帝】的支援卡片。
* 其效果可以用作回收帝王系列卡片。
===== 卡片資訊 =====
* 效果怪獸<br>等級3／暗屬性／網域族／攻擊力800／守備力1000
**【條件】在召喚了這張卡的回合內，將自己墓地裡2隻守備力1000點的怪獸返回牌組才能發動。
**【效果】對手將自身墓地裡1張魔法卡覆蓋到自己場上。然後可以將自己墓地裡1隻等級5以上且守備力1000點的怪獸加入手牌。
{{-}}

==== 賢帝近衛兵（{{ruby|賢帝近衛兵|けんていこのえへい}}） ====
* 於KP-17登場
* 效果完全致敬《炎帝近衛兵》，主要是減少召喚進化帝的費用。
** 但這張卡需要返回的怪獸數量更多，可能比《炎帝近衛兵》稍微難用一點。
===== 卡片資訊 =====
* 效果怪獸<br>等級3／暗屬性／網域族／攻擊力1200／守備力1000
**【條件】若自己牌組存在至少10張卡牌，從自己墓地裡守備力1000點的怪獸當中，將4隻攻擊力不是1200點的怪獸返回牌組才能發動。
**【效果】自己抽1張牌。然後這張卡的等級上升2，直到回合結束為止。
{{-}}

==== 賢帝 威廉（{{ruby|賢帝|けんてい}}ヴィルヘル） ====
* 效果可確定對方卡片，但只能破壞魔法卡，不能破壞陷阱卡。
===== 卡片資訊 =====
* 效果怪獸<br>等級5／暗屬性／網域族／攻擊力2000／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】確認對手場上1張裡側表示的魔法卡或陷阱卡。若那張卡是魔法卡，則可以接著破壞那張卡。
{{-}}

==== 才賢帝 威廉（{{ruby|才賢帝|さいけんてい}}ヴィルヘル） ====
* 【賢帝】的進化版。
* 效果可破壞任何卡片，但要由對方選擇。
===== 卡片資訊 =====
* 效果怪獸<br>等級8／暗屬性／網域族／攻擊力2400／守備力1000
** 可以只釋放1隻等級5以上且守備力1000點的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】對手破壞自身場上1張卡牌。
{{-}}

=== 星帝系列 ===
* 收錄於補充包「閃光的忘卻龍」，超速決鬥加入的新成員之一，是銀河族的成員。
* 和【賢帝】一樣有著破壞卡片的能力，但由於不用代價，效果綁了一堆限制。
<br>
==== 星帝家臣 赫貝（{{ruby|星帝家臣|せいていかしん}}ヘーベ） ====
* 為【星帝】的支援卡片。
* 其效果可以用作回收帝王系列卡片，同時召喚足夠素材召喚帝皇。
===== 卡片資訊 =====
* 效果怪獸<br>等級3／光屬性／銀河族／攻擊力800／守備力1000
**【條件】在召喚了這張卡的回合內，將自己墓地裡2隻守備力1000點的怪獸返回牌組才能發動。
**【效果】對手特殊召喚自身墓地裡1隻等級6以下的效果怪獸。然後自己可以將自己墓地裡1隻等級5以上且守備力1000點的怪獸加入手牌。
{{-}}

==== 星帝 艾斯托姆（{{ruby|星帝|せいてい}}エストローム） ====
* 效果可破壞對方怪獸，但只限等級6或下的效果怪獸，代表不能破壞等級6以上，或是任何通常怪獸。
===== 卡片資訊 =====
* 效果怪獸<br>等級5／光屬性／銀河族／攻擊力2000／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】破壞對手場上1隻第級6以下的效果怪獸。
{{-}}

==== 煌星帝 艾斯托姆（{{ruby|煌星帝|こうせいてい}}エストローム） ====
* 【星帝】的進化版。
* 效果可破壞任何卡片，但要由對方選擇。
===== 卡片資訊 =====
* 效果怪獸<br>等級8／光屬性／銀河族／攻擊力2400／守備力1000
** 可以只釋放1隻等級5以上且守備力1000點的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】對手破壞自身場上1張卡牌。
{{-}}

=== 龍帝系列 ===
* 收錄於預組牌組「帝王的凱旋」，超速決鬥加入的新成員之一，是幻龍族的成員。
* 效果各異，但共通條件是要求額外牌組中沒有卡牌。
<br>

==== 龍帝家臣 多拉黑（{{ruby|竜帝家臣|りゅうていかしん}}ドラーヘ） ====
* 【龍帝】的支援卡片。
* 其效果除了減少召喚進化帝的費用外外，也可以用作回收【龍帝】系列卡片。
===== 卡片資訊 =====
* 效果怪獸<br>等級3／水屬性／幻龍族／攻擊力800／守備力1000
**【條件】若自己額外牌組中沒有卡牌，將牌組最上方的卡牌送去墓地才能發動
**【效果】這張卡的等級上升2，直到回合結束為止。若自己在這個回合沒有特殊召喚過怪獸，則可以接著將自己墓地裡1隻「幻龍帝 林德魯姆」或「龍帝 林德魯姆」加入手牌
{{-}}

==== 龍帝 林德魯姆（{{ruby|竜帝|りゅうてい}}リンドルム） ====
* 效果除了可以破壞對方堆墓外，也可以減少對方場上怪獸的攻擊力，而且效果的唯一限制是額外牌組沒有任何卡片，是新【帝王】中比較好用的一張。
===== 卡片資訊 =====
* 效果怪獸<br>等級5／水屬性／幻龍族／攻擊力2000／守備力1000
** 可以只釋放1隻等級5以上且守備力1000點的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合內，若自己額外牌組中沒有卡牌才能發動。
**【效果】令對手墓地裡1張或2張卡牌返回牌組。然後可以將對手場上1隻效果怪獸的攻擊力與守備力下降800點，直到回合結束為止。
{{-}}

==== 幻龍帝 林德魯姆（{{ruby|幻竜帝|げんりゅうてい}}リンドルム） ====
* 【龍帝】的進化版。
* 效果可令任卡片回到牌組，但要由對方選擇，而且多了額外牌組清零的條件
===== 卡片資訊 =====
* 效果怪獸<br>等級8／水屬性／幻龍族／攻擊力2400／守備力1000
** 可以只釋放1隻等級5以上且守備力1000點的怪獸就升級召喚這張卡。
**【條件】在召喚了這張卡的回合內，若自己額外牌組中沒有卡牌才能發動。
**【效果】由對手挑出自身場上2張卡牌，自己令其中1張返回牌組。
{{-}}

=== 原六帝系列 ===
* 現時超速決鬥已引入《雷帝》《炎帝》《風帝》《地帝》《冰帝》《邪帝》
* 由於超速決鬥沒有召喚時誘發效果，因此全都改成效果只能在召喚回合才能發動。
* 由於每張的攻擊力高達2400，而且所有效果都沒有發動代價，甚至能夠達到一換一甚至一換二的效果，因此每張都被列入傳說卡
<br>
==== 雷帝 扎博爾格（{{ruby|雷帝|らいてい}}ザボルグ） ====
* 收錄於補充包KP13，為第一隻收錄的六帝，也是「六帝」怪獸中唯一一隻等級5怪獸。
* 由新效果不是召喚時必發，改善原版[[自爆]]的弱點。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級5／光屬性／雷族／攻擊力2400／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】破壞對手場上1隻怪獸。
{{-}}

==== 冰帝 梅比烏斯（{{ruby|氷帝|ひょうてい}}メビウス） ====
* 收錄於補充包「真・傳說覺醒包」。
* 效果和原版一樣，可以破壞場上1到2張魔陷。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／水屬性／水族／攻擊力2400／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】破壞對手場上1張或2張魔法卡及陷阱卡。
{{-}}

==== 炎帝 泰斯塔羅斯（{{ruby|炎帝|えんてい}}テスタロス） ====
* 初登場為V Jump特典卡，後期正式收錄於補充包「5周年包」。
* 和原版一樣，隨機丟棄對手的一張手牌並造成傷害，但因遊戲機制很多時候對手跟本沒有手牌可以弄。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／炎屬性／炎族／攻擊力2400／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】對手隨機丟棄1張手牌。若丟棄了怪獸，則給予對手那隻怪獸的等級×100點的傷害。
{{-}}

==== 地帝 格蘭瑪格（{{ruby|地帝|ちてい}}グランマーグ） ====
* 收錄於補充包「覺醒的爆速暴龍」。
* 和原版一樣，可以破壞一張蓋卡，不論是魔陷卡還是怪獸卡。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／地屬性／岩石族／攻擊力2400／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】破壞場上1張裡側表示卡牌。
{{-}}

==== 風帝 瑞薩（{{ruby|風帝|ふうてい}}ライザー） ====
* 初收錄於預組牌組「神鷹女郎姊妹」。
* 可以將場上一張卡片彈回牌組最上方。在原版為極強力的效果，但在超速決鬥因每回合可以抽多於一張所以影響較少。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／風屬性／鳥獸族／攻擊力2400／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】令場上1張卡牌返回牌組上方。
{{-}}

==== 邪帝 蓋烏斯（{{ruby|邪帝|じゃてい}}ガイウス） ====
* 收錄於預組牌組「帝王的凱旋」
* 由於超速決鬥始終沒有「除外」這項處理，因此效果改為回到牌組，但保留指定闇屬性怪獸就能向對手造成傷害的效果。
===== 卡片資訊 =====
* 效果怪獸（LEGEND）<br>等級6／闇屬性／惡魔族／攻擊力2400／守備力1000
**【條件】在召喚了這張卡的回合內才能發動。
**【效果】令場上1張卡牌返回牌組下方。若那張卡是闇屬性怪獸，則給予對手1000點傷害。
{{-}}

== 相關陷阱 ==

=== 升級衝擊（アドバンス・インパクト）===
* 收錄於補充包KP13，弱化版的《烈旋》
* 除了要求場上沒有怪獸，而且要以手牌作為代價才可以令對方怪獸成為自己的祭品。
===== 卡片資訊 =====
* 通常魔法
**【條件】若自己場上沒有怪獸才能發動。
**【效果】根據對手場上1隻怪獸的等級，將自己手牌中的怪獸送去墓地，直到其等級合計與那隻對手怪獸相同。這個回合，自己若為升級召喚而將釋放怪獸，可以釋放那隻對手怪獸來代替釋放自己場上1隻怪獸。
{{-}}

=== 皇帝領域（エンペラー・レルム）===
* 收錄於補充包KP13，弱化版的《真帝王領域》
* 效果被斬到只留下增加攻擊力效果，而且並不是永續效果，也是在RD中少數有著發動條件的場地魔法
* 卡圖表示《賢帝》和《星帝》為戰友關係<s>，難怪兩人效果如此相似</s>
===== 卡片資訊 =====
* 場地魔法
**【條件】必須將牌組上方2張卡牌送去墓地才能發動這張卡。
**【效果】只要這張卡以表側表示存在於場地區域，場上所有等級5以上且原本守備力1000點以上的怪獸的攻擊力上升400點。
{{-}}

=== 帝王的烈旋（帝王の{{ruby|烈旋|れっせん}}）===
* 收錄於預組牌組「帝王的凱旋」。
* 引入後變回通常魔法，但效果複製原版的效果。
* 對比起條件複雜的《升級衝擊》，這張卡的效果近乎無任何代價，因此被列入LEGEND卡之列。
===== 卡片資訊 =====
* 通常魔法（LEGEND）
**【條件】在自己沒有特殊召喚過額外牌組中的怪獸的回合內才能發動。
**【效果】挑選對手場上1隻怪獸。這個回合，自己若為升級召喚而將釋放怪獸，則可以釋放那隻對手怪獸來代替釋放自己場上1隻怪獸。發動這個效果後，自己不能特殊召喚額外牌組中的怪獸，直到回合結束為止。
{{-}}

=== 越境的帝王（{{ruby|越境|えっきょう}}の帝王）===
* 收錄於預組牌組「帝王的凱旋」，回收墓地之用
* 跟據卡圖所示，RD的新帝王和OCG的舊帝玊存在於不同的時空，但一次意外，暗帝入侵了RD的時空……
===== 卡片資訊 =====
* 通常魔法
**【條件】無
**【效果】從自己墓地裡攻擊力2000點與2400點的怪獸當中，將1隻守備力1000點的怪獸加入手牌。
{{-}}

=== 帝王的凱歌（帝王の{{ruby|凱歌|がいか}}）===
* 收錄於預組牌組「帝王的凱旋」，抽濾之用
* 卡圖為《光帝》和《闇帝》
===== 卡片資訊 =====
* 通常魔法
**【條件】若自己額外牌組中沒有卡牌，從自己手牌中守備力1000點的怪獸當中，將1隻攻擊力2000點或2400點的怪獸送去墓地才能發動。
**【效果】自己抽2張牌。然後可以令對手墓地裡1張或2張卡牌返回牌組。
{{-}}

=== 帝王的機才（帝王の{{ruby|機才|きさい}}）===
* 收錄於補充包KP13
* 圖中為《賢帝 威廉》
===== 卡片資訊 =====
* 通常陷阱
**【條件】若自己場上存在等級5以上且守備力1000點的怪獸，在對手召喚了怪獸時才能發動。
**【效果】令那隻怪獸轉為裡側守備表示。然後可以將自己墓地裡1隻等級5以上且守備力1000點的怪獸加入手牌。
{{-}}

=== 帝王的煌誕（帝王の{{ruby|煌誕|こうたん}}）===
* 收錄於補充包KP13
* 圖中為《星帝 艾斯托姆》
===== 卡片資訊 =====
* 通常陷阱
**【條件】在對手召喚或特殊召喚了等級8以下的怪獸十，將自己場上1隻怪獸送去墓地才能發動。
**【效果】以攻擊表示特殊召喚自己手牌中1隻，等級5以上且守備力1000點的怪。然後可以破壞對手那次召喚或特殊召喚的其中1隻怪獸。
{{-}}
