*置於桌面上執行的圖形介面程式，由'''核心程式'''、'''人格'''和'''外殼'''三個部分所組成。
*用途有美化桌面，代理執行其他程式，改變電腦使用者心情…等等。
*是桌面萌化用軟體之一。

== 前言 ==
* Windows98、撥接網路當紅時代有出現所謂的桌面小寵物，例如小綿羊、皮卡丘。
* 偽春菜就是與其類似的桌面小程式，但所含功用、程式複雜度以及其魅力遠遠超過小綿羊所有。
** 中文俗稱桌面小助理、桌面精靈、人格程式、桌面程式…等等諸如此類，族繁不及備載。不過了解偽春菜的人還是常以日文 '''伺か''' 或偽春菜稱之。
** 日本名為'''偽春菜'''、'''何か'''(なにか, nanika)、'''任意'''、'''さくら'''(sakura)，目前的正式名稱為'''伺か'''(うかがか, ukagaka)，這些會在源由中做說明。
* <s>不是假的[[池澤春菜]]</s>

== 源由 ==
*首先由此構想的是日本國立舞鶴工業高等専門學校的中西巧(真名)，他在1993年的在學中提出了假想人格的構想，也就是桌面AI秘書程式。
*到了1995年，他將這個人格程式引擎命名為「春菜」，這就是原始本名的由來，並在同年加入「翼システム」公司。
*然後同年與朋友發動了「第一次春菜 Project」，內容為設計好基本設計與設定，開始著手進行開發計畫。「ペルソナウェア(PersonaWare)」為其核心程式的代稱。
*在經歷了1996年的「第二次春菜 Project」之後開始受到外界矚目。
<br>
*1998年，程式開發已略成熟，中西巧將「ペルソナウェア(PersonaWare)先行公開版」開放給外界下載，同時他的高專時代朋友「野村」也開始起草春菜的原畫。
**次年，新版本「ペルソナウェア with "春菜"」以免費的方式公開。
**2000年，翼システム成立了子公司「Praesens」。
**同年中西巧因為作品大受歡迎而決定由免費軟體改為共享軟體，試用期為 30 日。
<br>
*但是改為收費之後並沒有使春菜有進一步的成長，主要原因是：
# 收費
# 免費版與收費版不相容
# 改為收費版後開發遲緩
*使得許多人對春菜感到失望而放棄使用新版本，有些人則轉向繼續使用或開發免費的舊版本ペルソナウェア(PersonaWare)。
<br>
*在許多不滿的使用者中當有一個人名為「黑衣鯖人(暱稱)」，他原先是非常看好ペルソナウェア(PersonaWare)，但在實際用過數日後卻感到失望不滿。
**加上新版本開發遲緩的關係，黑衣鯖人決心從原點再次開發核心程式。
<br>
*2000年5月25日黑衣鯖人公開他的作品，且將他的作品命名為「偽ペルソナウェア with "春菜"」以諷剌翼システム公司的阻礙春菜成長。
**當時的shell是由Saxyun(暱稱)所繪製，預設人格名為「さくら」及其使魔「うにゅう」，這就是「偽春菜」、「さくら」別名的由來。
**之後偽春菜的更新速度遠遠超過春菜，每天都有新功能。偽春菜擁有春菜所沒有功能，設計原創角色也比春菜更加平易近人，因此在這之後偽春菜的討論社群愈來愈熱門。
<br>
*2000底，黑衣鯖人將偽春菜改名為「あれ以外の何か with 偽春菜」以避免著作權等問題，因為當時 Praesens 公司正在為「ペルソナウェア with 春菜」申請專利。
*2001初，翼システム以及 Praesens 兩間公司的社長對黑衣鯖人提出了終止公開「あれ以外の何か with 偽春菜」的要求，包括網頁中與ペルソナウェア、春菜相關的名稱也必需停止使用，並宣*告若在一週內沒有回應，將採取強硬措施。
*此舉使得黑衣鯖人又將偽春菜再改名為「あれ以外の何か with 任意」，這就是「任意」的由來。
*不過就在一星期後，「あれ以外の何か with 任意」即停止公開，且網站在一個月後就關閉了，引起外界一陣嘩然。
<br>
*同一時間，「Doichan！」發表了基於「Sakura 語法」而製作出的核心程式「SAKURA Script Player」(SSP)。
*之後「あれ以外の何か with 任意」的開發轉為「閑馬永空(暱稱)」接手，並將其改名為「何か。（仮）」後重新發佈。
*其中預設人格「さくら」外型經過更改，已和春菜不相似，さくら著名的雙馬尾髮型，也是在這時期改成短髮。
<br>
*自此之後，Praesens 公司不再與偽春菜的開發有任何瓜葛，偽春菜就在此後蓬勃發展至今。
<br>
*同年八月，再次發布並正式訂名為「何か」，此為「何か」名稱的由來。
**隔年2002年三月，TanisELF(暱稱)成立了中文偽春菜後援會(Chinese Ukagaka Club，簡稱 CUC)
**四月，閑馬永空的本家「何か」網頁關閉，後來還傳出閑馬永空死亡的消息，是真是假目前不得而知。
**不過此次事件所造成的影響也不小，同好間又再次開始擔心偽春菜的存續問題，也接連影響到中文偽春菜後援會。
<br>
*同一時間，川上新夜(暱稱-舊)(今：ls)再度接手，將完成至 period 563 的版本開發至583，並將563 版以後的名稱改為「伺か」，這就是目前的正式訂名了。
*在發佈了583版之後，ls宣告停止開發，理由是「原為一應用軟體，而現已轉變為平台」，本家歷史至此告一段落。
**(ls原文：「一アプリケーションがプラットフォームになった為」)
*即使如此，其他核心的本體仍在不斷發展當中…

== 構造 ==
*偽春菜的構造很簡單，分別由三個部分所組成：核心程式(本體)、人格(ghost,ゴースト)和外殼(shell,シェル)。
*以一台電腦來比喻，本體就如同作業系統，而人格就是在其作業系統上執行的應用程式，外殼當然指的是我們看得到的圖型介面。

=== 核心程式(本體) ===
*在多年的發展下，核心程式日趨多元，當然不是所有的核心都適用於 Windows，
**以下列出常見的核心：

==== Windows ====
*'''Materia'''
**本家的核心，目前已停止開發。
*'''SSP (SAKURA Script Player)'''
**目前的主流核心，且僅此一核心有支援中文(繁簡皆有)，分別為TSSP,CSSP，目前仍在進化當中…
*'''C.R.O.W.(クロウ、The Character Requested Operation Ware)'''
**已經落沒的核心，在去年已停止開發。

==== Mac OS ====
*'''何か on 林檎'''
**適用於OS9之前版本)
*'''偽林檎(OS X)'''
**由於林檎(りんご,ringo)是日文蘋果的意思，意思當然指在蘋果電腦上的偽春菜，仍在發展當中…

==== UNIX 系 OS ====
*'''ninix'''
**命名取 Linux 諧音，現在開發休止中。
*'''ninix-aya'''
**比原來的 ninix 新加入許多功能，目前正發展當中…

=== 人格(ghost,ゴースト) ===
* ghost指的就是人工AI的部分，包含了許多的程式碼、對話句、時間指令…
* 通常在說到ghost的時候，也同時包含了shell在裡面，因為一個ghost需要shell來呈現。
* 因此下載ghost時常會將與ghost一起計畫的shell同捆在一起，如此一來在取得ghost的同時就可立即使用該ghost。
* 而該shell即被稱作default shell(預設介面)。
* 截至目前為止，人格的種類數量已經多到數不清了，各種屬性都有，毒舌、捏他…

=== 角色介面(shell,シェル) ===
* shell指的就是我們在螢幕上看到的人物外型，由同一伺か企畫的畫師負責。
* 而除了主要伺か角色之外，另外通常還會再搭配一個使魔，功用是與主角對話，亦或者是吐嘈主角…等等。
* 使魔的設定從何而來不得而知。
* 同一個ghost可以有很多種shell。
* 與ghost一樣，shell的種類也很多，從蘿莉到兄貴應有盡有…

=== 功能 ===
* 實際上可用功能隨ghost的不同而不同。
* 網路檢索、天氣預報、對時、便條、擲骰子、計時器、查 IP、清除計錄和資源回收筒、收取 RSS 新聞、玩遊戲、網路本體更新、檢查郵件、重新開機、關機…
* 而在心理上有調整心情、解除壓力…等等
* 視覺上則有<s>美化</s>萌化桌面的功用。
* 新版的核心可藉由移動滑鼠來進行觸摸特定部位的動作，通常有頭、臉、胸部。
